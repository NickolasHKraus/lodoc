Each directory contains an `index.md` file comprising the content of the section.

```
content
└── 01_getting-started
   └── index.md
```

An `index.md` has the following frontmatter:

```yaml
---
title:
url:
weight:
---
```

### Homepage

The content for the homepage is provided by `_index.md`.
